This project was developed as part of the Modern Application Development – II coursework and focused on building a complete grocery store platform using a Vue.js frontend and Flask backend following the MVC architecture pattern.

The application supports multiple user roles with role-based access control using Flask-Security. REST APIs were implemented using Flask-RESTful for handling product management, authentication, shopping cart operations, and order workflows.

Key features included:
- User authentication and RBAC with Flask-Security
- Category, product, and shopping cart management
- RESTful CRUD APIs using Flask-RESTful
- Responsive UI for both desktop and mobile devices
- Redis caching with Flask-Caching for improved performance
- Scheduled jobs and daily reminder systems using Celery
- Monthly HTML email reports and CSV export support

The backend was built with Flask, SQLite, SQLAlchemy, Redis, and Celery, while the frontend was developed using Vue.js and Vue Router. The project also included asynchronous task scheduling and email automation to simulate production-style application workflows.

This project strengthened my understanding of full-stack application architecture, API design, authentication systems, background job processing, and performance optimization techniques.
