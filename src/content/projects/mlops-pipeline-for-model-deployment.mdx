This project focused on building a complete MLOps workflow covering model training, deployment, monitoring, and scalability using modern cloud-native infrastructure.

The pipeline automated CI/CD workflows using GitHub Actions and deployed a containerized inference service on Kubernetes with Google Kubernetes Engine (GKE). Autoscaling support was configured to improve reliability under varying traffic loads.

Key features included:
- CI/CD automation with GitHub Actions
- Dockerized inference service deployment on GKE
- Kubernetes autoscaling and load testing
- Latency tracking and request logging
- SHAP explainability, fairness evaluation, and data drift detection

The project emphasized operational aspects of machine learning systems, including observability, deployment reproducibility, and production monitoring. It also provided hands-on experience with Kubernetes orchestration and scalable ML infrastructure design.
