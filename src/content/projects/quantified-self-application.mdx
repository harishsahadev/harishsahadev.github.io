This project was developed as part of the Modern Application Development – I coursework and focused on building a tracker application for managing and visualizing personal activity data.

The application allows users to create custom trackers, log entries, and monitor trends through dynamically generated visualizations. It was built using Flask with Jinja2 templating and Bootstrap 5 for responsive UI design.

Key features included:
- User authentication and session management
- CRUD operations for trackers and logs
- Form validation using Flask-WTF
- Dynamic graph generation and data visualization
- Responsive UI using Bootstrap 5

The backend handled tracker management and logging workflows, while visualization components helped users analyze tracked data over time. The project emphasized structured application design, server-side rendering, and interactive data handling.

This project strengthened my understanding of Flask-based web development, templating systems, authentication workflows, and data visualization techniques in full-stack applications.
