This project focused on predicting the total amount paid by travellers for taxi rides using a large dataset containing approximately 175,000 records and 17 features.

The workflow involved extensive exploratory data analysis (EDA), feature preprocessing, and training multiple machine learning regression models using Python and scikit-learn. Different algorithms were evaluated and compared to identify the best-performing approach for fare prediction.

Key features included:
- Exploratory Data Analysis (EDA) and feature preprocessing
- Regression model comparison using multiple ML algorithms
- Hyperparameter tuning with GridSearchCV
- Model evaluation using R² score and regression metrics
- Large-scale dataset handling and performance analysis

Several regression algorithms were tested, including Linear Regression, Decision Trees, SGD Regressor, KNN, Random Forest, AdaBoost, and XGBoost. The best performance was achieved using XGBoost with GridSearchCV hyperparameter tuning, obtaining an R² score of 0.9449.

This project improved my understanding of regression workflows, feature engineering, model evaluation, and hyperparameter optimization for large real-world datasets.
